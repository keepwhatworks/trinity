Serve this kit

This launchpad kit needs a local server.

It uses React with in-browser Babel, which browsers block over file://. Run it from a quick local server instead:

cd design-system/ui_kits/launchpad
python3 -m http.server 8000
# then open http://localhost:8000

(The website kit — ui_kits/website/ — is pure HTML/CSS and opens by double-click.)
